# Recruitment-analytics-dashboard

A comprehensive data analytics project to optimize the hiring process by analyzing recruitment metrics, candidate data, and identifying key bottlenecks.

## üéØ Project Overview

This project analyzes recruitment data to provide actionable insights for HR teams. It tracks key metrics like time-to-hire, cost-per-hire, source effectiveness, and diversity metrics.

### Business Problem
- High recruitment costs
- Long time-to-hire
- Ineffective sourcing channels
- Lack of diversity insights
- No data-driven hiring decisions

### Solution
A comprehensive analytics dashboard that:
- Tracks recruitment funnel metrics
- Identifies best-performing sources
- Analyzes time and cost efficiency
- Monitors diversity and inclusion
- Provides actionable recommendations

## üîç Key Metrics Analyzed

1. **Overall Metrics**
   - Total candidates processed
   - Hiring rate
   - Rejection rate
   - Average time-to-hire
   - Average cost-per-hire

2. **Source Effectiveness**
   - Conversion rate by source
   - Cost comparison across sources
   - Quality of candidates by source

3. **Department Analysis**
   - Hiring by department
   - Average salary offered
   - Time-to-hire by department

4. **Diversity Metrics**
   - Gender distribution
   - Location-wise hiring
   - Experience level distribution

5. **Candidate Funnel**
   - Application to hire conversion
   - Drop-off points
   - Status distribution
